# Polskie Radio Kivy

Prosta aplikacja do słuchania polskich stacji radiowych (RMF FM, Radio ZET, Antyradio itp.) napisana w Kivy + KivyMD + ffpyplayer.

Działa na desktopie (macOS/Linux/Windows), w trakcie pracy nad APK na Androida.

## Wymagania lokalne
pip install kivy kivymd ffpyplayer pyradios

## Uruchomienie
python main.py

## Status
- Desktop: RMF FM działa, pauza/stop bez crasza (po poprawkach)
- Android: w budowie (problemy z ffpyplayer w niektórych streamach)
